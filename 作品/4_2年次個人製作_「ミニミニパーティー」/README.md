\# ミニミニパーティー



\## 概要

C++とNintendo Switchを用いて制作した、3種類のミニゲームを収録したパーティーゲームです。



任天堂作品にみられる「本編にも負けないくらい面白いミニゲーム」をコンセプトに制作しました。

短時間でも繰り返し遊びたくなるようなゲーム体験を目指し、3つのミニゲームを実装しました。

\## 工夫した点

Nintendo Switch向け開発の都合上、実機確認には制約がありました。

そのため、コードレビューや設計の見直しを繰り返し、不具合の発生を未然に防ぐことで効率的に開発を進めました。



また、夏休み期間中も学校へ通い、仕様設計から実装、調整まで一貫して行い、3種類のミニゲームを完成させました。



\## 学んだこと

限られた検証環境の中でも品質を担保するためには、事前の設計やレビューが重要であることを学びました。



また、個人制作を通して、企画から実装、デバッグまでを一人で進める計画力と継続力を身につけました。

開発環境の都合上、ソースコードは載せられません
