# 操糸奇譚（そうしきたん）

## 概要
8人チーム「SMD（Super Mizu Dragon）」で制作した、糸を使って敵を倒しながら巫女を守るパズルアクションゲームです。

プレイヤーは糸を張り巡らせて敵を囲み、繭にすることで敵を倒します。
さらに、繭を糸で貫くことで周囲の敵の移動速度を低下させ、糸に毒属性を付与し、継続ダメージを与えることができます。

巫女を誤って囲んで繭にしてしまう、または巫女が敵から3回攻撃を受けるとゲームオーバーとなります。
