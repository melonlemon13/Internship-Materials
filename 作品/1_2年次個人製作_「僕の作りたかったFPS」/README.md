# 僕の作りたかったFPS

## 概要
C++とDirectX11を用いて制作した、一人用の3D FPSゲームです。
プレイヤーは銃やスキルを駆使して敵を倒し、制限時間内のスコア更新を目指します。

## プレイ動画
https://www.youtube.com/watch?v=Ro2AyLyTZl0

## 使用技術
- C++
- DirectX11
- XAudio2
- Assimp
- Git / GitHub

## 実装内容
- オブジェクト管理システム
- ロード画面を用いた段階的なアセット読み込み
- 武器システム（ハンドガン・アサルトライフル・ナイフ）
- 敵AI
- 衝突判定
- スキルシステム（ブリンク・グレネード・アルティメット）
- UIシステム
- サウンドシステム

## 工夫した点
保守性を意識し、オブジェクトの生成・削除をキューで管理する仕組みを実装しました。
また、ロード処理を複数フレームに分割することで、ロード画面を表示したまま処理を進められるよう設計しました。

## 学んだこと
機能追加を繰り返す中で、後から拡張しやすい設計を意識することの重要性を学びました。
また、問題に直面した際には自ら解決策を調査し、自分の環境に合わせて実装・検証を繰り返す力を身につけました。
