\# めんへらめりぃちゃん



\## 概要

5人チームで制作した、「断罪」をテーマにしたノベルゲームです。



プレイヤーは、めんへらのめりぃちゃんを刺激しないようにメッセージを選択しながら、めりぃちゃんとの関係の断絶を目指します。



学内企画として、3日間（実作業2日間）で制作しました。

\## 工夫した点

初めてUnity6を用いた制作であり、限られた期間で完成させる必要がありました。

そのため、優先順位を決めながら、実装工数が少なく、バグが発生しにくい方法を選択して開発を進めました。



また、チームリーダーとしてメンバーの進捗を把握し、状況に応じてタスクの調整や優先順位の変更を行い、短期間での完成を実現しました。



\## 学んだこと

短期間のチーム開発を通して、限られた時間の中で優先順位を付けて判断する力や、タスク管理・進行管理の重要性を学びました。



また、リーダーとして、メンバーの状況を把握しながら意思決定を行う力や、短期間でも完成度を意識して開発を進める重要性を学びました。
